# Hospitality-data-dashboard
A complete hospitality analytics project using Pandas, NumPy, Plotly to analyze hotel booking trends and build an interactive dashboard.
# 🏨 Hospitality Data Dashboard

## 📌 Overview
This project analyzes hotel booking data to uncover trends in revenue, customer preferences, and occupancy rates.

## 🔍 Key Insights
- Which city generates highest revenue
- Most profitable room type
- Monthly booking trends
- Price vs rating relationship

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Plotly

## 📊 Dashboard Features
- Interactive graphs
- Dropdown filters
- Real-time data insights

## 🚀 How to Run
```bash
pip install -r requirements.txt
python dashboard/app.py
